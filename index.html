What is this?

This is the development home for the Android port of the wildly popular and insanely ever-lasting game Neverball. Neverball is a ball-rolling game originally developed for desktops. Neverball ME ("mobile edition") refers to its mobile variants. Neverball is GPLv2, requires SDL 1.2, and runs against OpenGL ES 1.1 specifications. It is available for devices using Android 2.1 (Eclair) and above; you can find it on the Android Market.

Originally, Neverball was developed by Robert Kooima, but over the years it has attracted a large number of contributors. The main Neverball SVN continues to grow with new features, bugfixes, and additional maps. As Neverball development continues, so too will Neverball ME. This port is maintained by one person.

Gameplay

In Neverball, your objective is to roll a ball through a course within a given time. If you can't do it, or fall off the track, you lose a ball.

Neverball ME uses the touchscreen and accelerometer of your phone. Tap the screen to select menu items. When you're playing the game, tilt the device left, right, forward, or backward, in order to roll the ball towards the coins and the goal.

That's right—coins. Most levels require you to collect a minimum number of coins before unlocking the goal:

  • Yellow coins are worth one coin
  • Red coins are worth five coins
  • Blue coins are worth ten coins

After getting 100 coins, you'll get an extra ball.

When you start Neverball for the first time, it downloads a 7z file filled with maps, sounds, and other necessary game data. This file is stored on the SD card, so there's no need to worry about running out of space. I recommend you switch to WiFi to grab the files quickly. As time passes I intend to introduce more maps, which will require additional downloads.

Compiling Neverball ME

As Neverball is GPL, the source code for the Android port is provided on GitHub. You can find the link to the project at the top of this site. The Android port relies heavily on an SDL port, also on GitHub. (I am indebted to pelya for this project.) For all information on building Neverball ME, refer to the GitHub page.

Please, don't be a jerk and redistribute an APK that provides no significant improvement to the game.

What is this?

Neverputt is a miniature golf game built on the same graphics and physics engine as Neverball. It is a separate game that shares many of the same features, code, and design implementations as Neverball.

Neverputt is GPLv2, requires SDL 1.2, and runs against OpenGL ES 1.1 specifications. It is available for devices using Android 2.1 (Eclair) and above; you can find it on the Android Market.

Best of all, the game supports multiple players!

Gameplay

If you know the basic rules of golf, Neverputt gameplay should be no surprise. Anywhere from one to four players can participate. You must get your ball into the goal in the shortest number of moves possible. The gameplay increases in difficulty as courses become longer and introduce new elements such as movable obstacles. Falling off the course counts against you as a stroke.

To change the direction you want to hit your ball, flick your finger left or right on the touchscreen. To control the strength of your hit, move your finger forward or backward. Pressing your device's Menu key hits the ball.

When you start Neverputt for the first time, it downloads a 7z file from the internet filled with maps, sounds, and other necessary game data. This file is stored on the SD card, so there's no need to worry about running out of space. I recommend you switch to WiFi or 3G/4G to grab the files quickly.
